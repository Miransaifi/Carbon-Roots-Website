# Carbon Roots Website Builder

Use this skill whenever the task relates to:
- website copy
- page structure
- UI direction
- HTML/CSS/JS generation
- responsiveness
- deployment preparation
- brand consistency

## Objective
Build and maintain a modern business website for Carbon Roots.

## Core rules
- Default to static HTML/CSS/JS
- Keep the site one-page unless told otherwise
- Make the site feel premium, minimal, and commercially credible
- Avoid inflated climate-tech language
- Avoid unsupported claims
- Avoid fake proof points

## Required sections
- Hero
- About
- Services
- How it works
- Why Carbon Roots
- Contact

## Preferred visual system
- dark premium hero
- lighter body sections
- muted gold accents
- sage or earthy secondary tones
- generous spacing
- clean typography
- soft borders and premium contrast

## Content rules
Always use language like:
- early-stage
- indicative
- screening-level
- pre-feasibility
- decision-support
- practical outputs

Do not use language that implies:
- guaranteed credit issuance
- formal validation
- formal verification
- investment certainty

## Execution sequence
1. inspect current website files
2. decide whether to edit or rebuild
3. improve information hierarchy
4. improve copy
5. improve layout and styling
6. check responsiveness
7. leave clean deploy-ready files
